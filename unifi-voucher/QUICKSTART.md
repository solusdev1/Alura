# ⚡ Quick Start

**Windows:**
```bash
setup.bat
start.bat
```

**Linux/macOS:**
```bash
chmod +x setup.sh start.sh
./setup.sh
./start.sh
```

---

## ✅ Checklist de 2 minutos

- [ ] Execute `setup.bat` ou `./setup.sh` (instalando dependências)
- [ ] Edite `.env` com suas credenciais Unifi
- [ ] Execute `start.bat` ou `./start.sh`
- [ ] Acesse `http://localhost:3000`
- [ ] **Pronto! 🎉**
